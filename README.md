# 🌱 CarbonWise - Carbon Footprint Awareness Platform

## 📌 Overview
CarbonWise helps users calculate, track, and reduce their carbon footprint through real-time insights and sustainability tracking.

---

## 🚀 Features
- Carbon Footprint Calculator
- Personalized Eco Insights
- Eco Badges (Green / Yellow / Red)
- Local Storage Tracking
- Testing Report Page
- Responsive UI Design

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- LocalStorage
- Netlify Deployment
